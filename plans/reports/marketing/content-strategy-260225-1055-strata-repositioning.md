# Content Strategy: Strata Repositioning

> **From:** "Personal Task Manager" → **To:** "AI Productivity Assistant"
> **Killer Feature:** System-wide text enhancement (Inline Enhance ⌘⌥E)
> **Target Audience:** Freelancers, AI Power Users, Privacy-conscious professionals

---

## Executive Summary

Strata's current positioning as a "Task Manager" fundamentally misrepresents its true value proposition. Code analysis reveals **AI enhancement features (2,400 LOC, 8.5/10 complexity)** significantly outweigh **task management (1,200 LOC, 4.0/10 complexity)**. The system-wide Inline Enhance feature is the only capability requiring Developer ID + Accessibility API — creating a unique competitive moat.

**Recommended Repositioning:**
- **Category:** AI Productivity Utility
- **Tagline:** "Your AI, anywhere on your Mac"
- **Lead with:** Inline Enhance, BYOK, Privacy
